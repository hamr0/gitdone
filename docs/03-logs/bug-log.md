# Bug Log

Track bugs found and their resolutions.

## Format

```
### Bug: [Title]
**Date**: YYYY-MM-DD
**Severity**: Critical | High | Medium | Low
**Symptom**: What was observed?
**Root Cause**: What caused it?
**Fix**: How was it resolved?
**Files**: Which files were modified?
```

---

*Add entries as bugs are discovered and fixed.*
